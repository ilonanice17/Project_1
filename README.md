## 📝 Описание программы
Expense Tracker — это десктопное приложение с графическим интерфейсом для учёта личных расходов. Программа позволяет быстро добавлять транзакции, фильтровать их по категориям и датам, а также автоматически подсчитывать общую сумму за выбранный период. Все данные сохраняются в формате JSON, что обеспечивает простоту резервного копирования и переноса между устройствами. Приложение написано на Python с использованием встроенной библиотеки tkinter и не требует установки сторонних зависимостей.

## 🚀 Как использовать
1. **Требования:** Установите Python 3.6 или выше. Библиотека `tkinter` уже встроена в стандартную поставку Python.
2. **Запуск:** Откройте терминал в папке с проектом и выполните команду: `python expense_tracker.py`
3. **Добавление расхода:** Введите сумму, выберите категорию из выпадающего списка, укажите дату в формате `ГГГГ-ММ-ДД` и нажмите кнопку `➕ Добавить расход`.
4. **Фильтрация:** Выберите нужную категорию, задайте диапазон дат в полях `С:` и `По:` и нажмите `🔍 Применить фильтр`. Таблица обновится, а итоговая сумма пересчитается.
5. **Подсчёт за период:** Укажите даты в полях `С:` и `По:`, затем нажмите `📊 Рассчитать за период`. В правом нижнем углу отобразится сумма расходов только за указанный интервал.
6. **Сохранение и загрузка:** Данные сохраняются автоматически при закрытии программы. Для ручного управления используйте кнопки `💾 Сохранить в JSON` и `📂 Загрузить из JSON`.
