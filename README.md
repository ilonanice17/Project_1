## 📖 Описание
**Expense Tracker** — это десктопное приложение для учёта личных расходов. Программа предоставляет удобный графический интерфейс для добавления транзакций, фильтрации данных по категориям и датам, а также автоматического подсчёта итоговых сумм за выбранный период. Все данные сохраняются в локальном файле `expenses.json`.

## ✨ Функционал
- ➕ Добавление расходов (сумма, категория, дата)
- 🔍 Фильтрация по категории и диапазону дат
- 📊 Подсчёт общей суммы за произвольный период
- 💾 Автоматическое сохранение и загрузка данных (JSON)
- ✅ Валидация ввода (положительные числа, корректный формат даты)
- 🖥️ Кроссплатформенность (Windows, macOS, Linux)

## 🛠 Требования
- Python `3.6` или выше
- Модуль `tkinter` (входит в стандартную библиотеку Python)
- Сторонние зависимости **не требуются**
