## 📝 Описание программы
Expense Tracker — это кроссплатформенное десктопное приложение с графическим интерфейсом для учёта личных расходов. Программа позволяет добавлять транзакции, фильтровать их по категориям и датам, а также автоматически подсчитывать итоговую сумму за выбранный период. Все данные сохраняются в формате JSON, что обеспечивает простоту резервного копирования и переноса.

## 🚀 Как использовать
1. **Требования:** Python 3.6 или выше (библиотека `tkinter` встроена в стандартную поставку Python).
2. **Запуск:** 
   ```bash
   python expense_tracker.py
